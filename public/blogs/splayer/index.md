## 📋 软件概述
> 🎶SPlayer是一款集多种功能于一身的开源音乐播放器，专为音乐爱好者设计。它不仅支持内置音源和用户自行导入的歌曲，还支持云盘导入，极大地丰富了用户的音乐库。SPlayer具有逐字歌词显示功能，使你能实时跟随每一句歌词，深入感受歌曲的每一个细节。
***

🎵 **丰富的音乐功能**
支持扫码登录、手机号登录、每日签到、云贝签到、私人FM、每日推荐歌曲、云盘音乐等完整功能

💻 **桌面歌词**
支持桌面歌词显示，可自定义样式、位置、字体大小等，支持逐字歌词和歌词翻译

📁 **本地音乐管理**
支持本地歌曲管理及分类，简易的本地音乐标签编辑及封面修改，支持播放部分无版权歌曲

🎨 **主题自适应**
封面主题色自适应，支持全站着色，Light / Dark / Auto 模式自动切换

⬇️ **下载功能**
支持下载歌曲/批量下载，最高支持 Hi-Res，需具有相应会员账号

🔄 **多种部署方式**
支持 Docker、Vercel、服务器部署，也可本地部署，提供完整的 API 接口和 WebSocket 控制
